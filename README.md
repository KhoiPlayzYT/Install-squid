# Install-squid
Installs squid on your computer.

Install command:
```bash
curl -fsSL https://raw.githubusercontent.com/KhoiPlayzYT/Install-squid/main/ubuntu.sh | sudo bash
```
or for all you wget enjoyers:
```bash
wget -qO- https://raw.githubusercontent.com/KhoiPlayzYT/Install-squid/main/ubuntu.sh | sudo bash
```

# What this script actually does:
1. Installs squid
2. Starts it and makes it start on boot

# Supported Linux distros so far:
- Ubuntu
